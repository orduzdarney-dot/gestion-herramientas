3. Acuérdate de esto para mañana:
La palabra mágica es "a": Si en el examen usas open("logs.txt", "w"), el profesor te bajará puntos porque "w" borra lo anterior. Usa "a" de Append (Acumular).

strftime: Úsalo para que la fecha se vea bonita: datetime.now().strftime("%Y-%m-%d %H:%M:%S").

No te compliques con el Nombre Completo: Si en la función donde estás no tienes el nombre del usuario (solo el ID), haz una búsqueda rápida en la lista de usuarios o simplemente pon el ID en el log. Lo importante es que el log se genere.

Consejo para la hora del examen:
Si te trabas buscando el nombre del usuario para el log, pon:
registrar_error_log(f"ID Usuario: {usuario['id']}", ...)
Es mejor entregar un log con ID que no entregar nada por perder tiempo buscando el nombre.
